Projects

  • Phylogenetic Inference Using Machine Learning on BRCA2: This project involves using machine learning(BiLSTMs MODEL) techniques to perform phylogenetic inference on the BRCA2 gene. The objective was to identify evolutionary patterns and variations across different species, focusing on understanding the genetic divergence and selection pressures acting on this critical gene associated with breast cancer susceptibility. The project was carried out under the guidance of Dr. Anindita Bhadra, combining computational biology approaches with machine learning algorithms to analyze sequence data and build evolutionary models.
