Projects

This project involves using machine learning techniques to perform phylogenetic inference on the BRCA2 gene. The objective was to identify evolutionary patterns and variations across different species, focusing on understanding the genetic divergence and selection pressures acting on this critical gene associated with breast cancer susceptibility. The project was carried out under the guidance of Dr. Anindita Bhadra, combining computational biology approaches with machine learning algorithms to analyze sequence data and build evolutionary models.

In this project, I explored codon usage bias—the preference for certain synonymous codons—in five diverse species (E. coli, S. cerevisiae, P. falciparum, C. elegans, A. thaliana). Using Python tools like Biopython, Pandas, Scikit-learn, and Matplotlib, I processed genomic coding sequences, calculated Relative Synonymous Codon Usage (RSCU) values, and applied statistical and machine learning techniques such as PCA and KMeans clustering. The analysis revealed species-specific patterns and evolutionary insights, visualized through heatmaps, dendrograms, and correlation plots. This work demonstrates how computational biology and data analysis can uncover meaningful patterns in genetic data and lays the groundwork for scalable future studies using expanded datasets and advanced models.
